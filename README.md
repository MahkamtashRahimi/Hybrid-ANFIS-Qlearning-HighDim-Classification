This project proposes a hybrid neuro-fuzzy classification framework for high-dimensional datasets, combining:
- LDA (Linear Discriminant Analysis): Reduces the dimensionality of data while preserving class separability.
- PSO (Particle Swarm Optimization): Selects the most relevant features.
- ANFIS (Adaptive Neuro-Fuzzy Inference System): Performs classification using a neuro-fuzzy approach.
- Q-learning: Dynamically tunes ANFIS parameters for improved performance.

The framework aims to efficiently handle high-dimensional data by reducing complexity while improving classification accuracy.
